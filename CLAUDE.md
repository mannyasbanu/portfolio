# Tech Stack

- **Next.js 16** - Latest version with breaking changes from Next.js 14/15. Read guides in `node_modules/next/dist/docs/` before writing code.
- **Tailwind CSS 4** - Utility-first CSS framework.
- **Framer Motion** - Animation library for React.

## Additional Dependencies

- `clsx` + `tailwind-merge` - Utility functions for conditional classes and class merging.
- `lucide-react` - Icon library.

## Project Structure

- `src/app/` - App router pages and layouts
- `src/components/` - Reusable components
